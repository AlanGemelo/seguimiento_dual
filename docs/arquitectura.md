# Arquitectura de Dashboards

El sistema cuenta con diferentes paneles de control (dashboards) según el rol del usuario:

- **dashboardSuperAdmin.blade.php**  
  Destinado a **Administradores y Directores**.  
  Funcionalidades: gestión de usuarios, control de roles, administración de convenios, reportes globales.

- **dashboard.blade.php**  
  Destinado a **Mentores Académicos Y Directores**.  
  Funcionalidades: seguimiento de estudiantes, revisión de convenios, reportes académicos.

- **dashboardEstudiantes**  
  Destinado a **Alumnos**.  
  Funcionalidades: consulta de estatus, documentos, becas, convenios y notificaciones personales.
